# Solaris

## Install

```sh
npm install
```

## Build & Launch

Start server:
```sh
npm run start
```

Build (no sourcemaps):
```sh
npm run build
```

Auto-build for development (with sourcemaps, etc):
```sh
npm run watch
```

Start server & watch for development:
```sh
npm run start-n-watch
```

## Development

```sh
npm run dev-watch
```
